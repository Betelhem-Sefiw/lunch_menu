# 🥗 Lunch Picker App

An interactive web application built with HTML, CSS, and Vanilla JavaScript that manages a dynamic lunch menu using array operations and DOM manipulation.

---

## ✨ Features

* **Add Items:** Insert new lunch items to either the start (`unshift`) or end (`push`) of the menu array.
* **Remove Items:** Delete items from either the start (`shift`) or end (`pop`) of the menu array.
* **Random Picker:** Select a random lunch item from the active menu using `Math.random()`.
* **Dynamic UI Updates:** Direct user feedback and updated menu state rendered to the DOM on every interaction.

---

## 🛠️ Built With

* **HTML5 (`lunch_menu.html`):** Page structure, inputs, and button controls.
* **CSS3 (`lunch_menu.css`):** Custom styling, layout, and button designs.
* **JavaScript (`lunch_menu.js`):** Array manipulation, event listeners, and DOM updates.

---

## 💡 Key Learnings

1. Connecting core JavaScript array methods (`push`, `pop`, `shift`, `unshift`) to visual HTML elements.
2. Handling user text inputs safely and preventing empty item additions.
3. Decoupling DOM event handling from primary business logic functions.
